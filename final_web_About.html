About This Project

The world is too big, and still we go back to the same familiar places over and over again. This project is trying to discover new destinations by the places that we are familiar with.
  • The data was collected from TripAdvisor website.
  • Each destination is defined by the top attractions suggested by TripAdvisor.
  • Using PCA and T-sne the data is presented on two axis.
  • Each country is represented as an image, which is one of top Google searching results for "country name + travel"
Enjoy!
